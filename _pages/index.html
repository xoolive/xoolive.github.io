Noteworthy open-source projects

  • xoolive/quadtree » a quadtree library for managing moving objects
  • xoolive/geodesy » a toolkit for geodesy computing (spherical and WGS84)
  • xoolive/facile » a Python wrapping for OCaml FaCiLe library (constraint programming)

I also keep an eye on the following open-source projects:

  • pandoc.org » pandoc is a tool to convert documents from one markdown language to another: it supports Markdown, reST, org-mode, HTML, LaTeX, docx, epub, among others.
  • jupyter.org » The Jupyter project develops a web application that allows to create and share documents that contain live code, equations, visualizations and explanatory text.
  • typerex.org » TypeRex is a set of tools and libraries for developing applications in OCaml.
