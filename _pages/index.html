Noteworthy open-source projects

I maintain the following small-scale open-source tools:

  • xoolive/quadtree » a quadtree library for managing moving objects
  • xoolive/geodesy » a toolkit for geodesy computing (spherical and WGS84)
  • xoolive/facile » a Python wrapping for OCaml FaCiLe library (constraint programming)

I also keep an eye on the following open-source projects:

  • jgm/pandoc » a tool to convert documents from one markdown language to another: it supports Markdown, reST, org-mode, HTML, LaTeX, docx, epub, among others. more
  • jupyter/jupyter » The Jupyter project develops a web application that allows to create and share documents that contain live code, equations, visualizations and explanatory text. more
  • www.typerex.org » TypeRex is a set of tools and libraries provided by OCamlPro for developing applications in OCaml. Since you are there, keep an eye on OCamlPro as well!
